## 📖 About the Project

This repository contains the source code for the **AIRA Study Centre** website. The website is designed to be a modern, professional, and mobile-responsive online presence for an education coaching institute. It highlights their core programs, teaching methodologies, success stories, and includes interactive elements to encourage parents to book free demo classes.

## 🚀 Technologies Used

The website was built from the ground up using core web technologies to ensure fast loading times, excellent SEO, and full control over the user interface without relying on heavy frameworks.

### 🌐 Core Technologies
* **HTML5:** Used for semantically structuring the web pages (Home, Programs, About Us, Results, Contact) and ensuring best accessibility practices.
* **CSS3 (Vanilla):** Custom styles crafted for a clean, modern aesthetic. Includes:
  * CSS Variables (Custom Properties) for unified theming and colors.
  * Flexbox and CSS Grid for structured, responsive layouts across screens.
  * Smooth transitions, micro-animations, and hover effects for a dynamic UI.
  * Media queries ensuring complete mobile responsiveness.
* **Vanilla JavaScript:** Implements interactive UI features such as the mobile navigation hamburger toggle, responsive sticky headers, and smooth scrolling functionality.

### 🎨 Design & Assets
* **Google Fonts:** Features `Libre Baskerville` for elegant, professional typography matching the brand identity.
* **SVG Graphics:** Lightweight vector graphics used for crisp WhatsApp floating buttons, menu toggles, and footer branding.

## 🗂️ Project Structure

* `Aira_website.html` - The main landing/home page highlighting the value proposition.
* `about.html` - Information about the institute's vision, mission, and the founders.
* `programs.html` - Detailed breakdown of coaching programs from Foundation to Board Mastery.
* `results.html` - Parent ratings, testimonials, and student success stories.
* `contact.html` - Contact details, location map info, and demo class booking.
* `style.css` - The master stylesheet governing the design system of the entire website.
* `script.js` - Client-side interactivity and Document Object Model (DOM) manipulation logic.
